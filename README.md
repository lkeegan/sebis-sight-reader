# Sightreader

A kid-friendly sight-reading web app that listens for notes and gives instant feedback.

## Run locally

1. Install dependencies:

```
npm install
```

2. Start the dev server:

```
npm run dev
```

3. Open the URL shown in the terminal (typically `http://localhost:5173`).

## Tests

```
npm test
```

## Build

```
npm run build
```

## Preview the build

```
npm run preview
```
